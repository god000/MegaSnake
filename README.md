# MegaRace Chaos

Браузерная мультиплеерная аркадная змейка на 5 игроков. Матчи по 3 минуты, цель — собрать максимальную длину. 25 бустов + Mystery Box, тороидальное поле, столкновения не убивают, авторитативный сервер на Node.js + ws, клиент на TypeScript + Canvas 2D.

## Структура

- `shared/` — типы, протокол, константы, каталог бустов.
- `server/` — Node.js + `ws`, авторитативный игровой цикл 20Hz.
- `web/` — Vite + TypeScript, простой клиент с Canvas 2D.

## Запуск (dev)

Из корня:

```bash
npm install
npm run dev
```

После этого:

- сервер слушает `ws://localhost:8787`
- клиент доступен на `http://localhost:5173`

Можно открыть несколько вкладок и присоединиться к одному матчу. Если живых игроков < 5, через 5 секунд после первого подключения слоты добиваются ботами.

## Управление

- Стрелки или WASD — движение.
- Развороты на 180° игнорируются.
- Цель — макс длина к концу таймера.

## Что уже работает

- Лобби на 5 игроков с автоматической добивкой ботами.
- Тороидальное поле 60×36.
- Спавн еды по тайм-кривой (бодрый старт → мясо к финалу).
- Все 25 бустов (`shared/src/boosts.ts`), включая временную широкую голову `Hammer Head`.
- Mystery Box: распределение 60% баф / 30% дебаф / 10% глобальный хаос.
- Анти-фрустрация: жёсткие дебафы не стакаются на одного игрока.
- Лидерборд + статусы активных эффектов в HUD.
- Отчёт после матча (длина, еда, бусты, mystery, кол-во смен лидера, разрыв 1↔5).
